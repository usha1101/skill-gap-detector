# 🚀 Smart Multi-Branch Career Skill Gap Detector

## 📌 Overview

A web-based application that helps students identify missing skills required for their target job roles across multiple engineering domains.

---

## 🌟 Features

* Multi-branch support (CS, Mechanical, ECE, EEE, VLSI, Civil)
* Role-based skill gap analysis
* Match percentage calculation
* Missing skills detection
* Learning roadmap generation

---

## 🛠 Tech Stack

* Python (http.server)
* HTML
* CSS
* JSON

---

## ⚙️ How It Works

1. Select your branch
2. Choose your target role
3. Enter your current skills
4. Get:

   * Match %
   * Missing skills
   * Learning roadmap

---

## 🚀 How to Run

```bash
python server.py
```

Open:
http://localhost:8000
